## A Bit to Decimal

|Bit | Decimal|
|--|--|
| 0 | 0 |
| 1 | 1 |

## Two Bit to Decimal

|Two Bit| Decimal|
|--|--|
| 00 | 0 |
| 01 | 1 |
| 10 | 2 |
| 11 | 3 |
