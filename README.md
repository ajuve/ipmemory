# IPmemory

A binary calculator for students who are learning binary conversion.

## Bit to Decimal

|Binary 1 (bit) | Decimal|
|--|--|
| 0 | 0 |
| 1 | 1 |

## Binary 2 to Decimal

|Binary 2| Decimal|
|--|--|
| 00 | 0 |
| 01 | 1 |
| 10 | 2 |
| 11 | 3 |
